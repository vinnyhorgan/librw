# librw workspace

We are building `rw` — a C99 reimplementation of a RenderWare-like 3D rendering API for a fantasy console, targeting OpenGL ES 2.0 with no extensions.

The work lives in `new_rw/`. See `new_rw/PLAN.md` for the full spec.

Everything outside `new_rw/` is the original librw C++ codebase and is **reference only**. Port algorithms from it, do not copy code.
